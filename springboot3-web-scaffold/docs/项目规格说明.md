# 项目规格说明（Spring Boot 3.0 简单 Web 脚手架）

## 1. 项目目标
提供一个可直接运行、可扩展、包含基础测试能力的 Spring Boot 3 Web 脚手架，用于：
1. 快速创建 REST 接口项目；
2. 作为教学或团队模板的初始化版本；
3. 后续扩展数据库、鉴权、日志、监控等模块。

## 2. 范围（Scope）
### 2.1 已实现范围
- 基础工程结构（Maven 标准目录）；
- 应用启动类；
- 两个演示接口：`/api/hello`、`/api/health`；
- 单元/集成测试（MockMvc）；
- 基础配置文件（`application.yml`）。

### 2.2 非本期范围
- 数据库访问（JPA/MyBatis）；
- 身份认证与授权（Spring Security）；
- 统一异常处理与全局日志链路追踪；
- 容器化部署（Docker/K8s）。

## 3. 技术规格
- **JDK**：17+
- **构建工具**：Maven 3.8+
- **框架**：Spring Boot 3.2.5（Spring Boot 3.x）
- **Web**：Spring MVC（spring-boot-starter-web）
- **测试**：spring-boot-starter-test，JUnit 5，MockMvc

## 4. 模块与目录规格
- `src/main/java/com/example/demo/DemoApplication.java`：应用入口。
- `src/main/java/com/example/demo/controller/HelloController.java`：示例 API。
- `src/main/resources/application.yml`：应用配置。
- `src/test/java/com/example/demo/controller/HelloControllerTest.java`：接口测试。
- `docs/项目规格说明.md`：规格文档。

## 5. 接口规格
### 5.1 GET /api/hello
- **用途**：返回简单欢迎信息。
- **请求参数**：无。
- **成功响应（200）**：
```json
{
  "message": "Hello Spring Boot 3",
  "version": "v1"
}
```

### 5.2 GET /api/health
- **用途**：返回服务状态信息。
- **请求参数**：无。
- **成功响应（200）**：
```json
{
  "status": "UP",
  "service": "springboot3-web-scaffold",
  "timestamp": "ISO-8601 时间字符串"
}
```

## 6. 运行与验证规格
### 6.1 本地运行
```bash
mvn spring-boot:run
```
默认端口 `8080`。

### 6.2 测试运行
```bash
mvn test
```
要求测试通过。

### 6.3 构建打包
```bash
mvn clean package
```
产物路径：`target/springboot3-web-scaffold-0.0.1-SNAPSHOT.jar`

## 7. 质量与扩展建议
1. 增加统一响应体与错误码规范；
2. 引入参数校验（`spring-boot-starter-validation`）；
3. 增加全局异常处理（`@RestControllerAdvice`）；
4. 添加 Actuator 与 Prometheus 指标；
5. 补充 API 文档（springdoc-openapi）。
